# PROJETO COTAÇÃO NO ARDUINO

### Axel de Oliveira Palmeira (1814768)
### Thiago de Andrade Prado (3821314)
### Victor Eduardo Rodrigues de Abreu e Silva (8576891)

Esse projeto consiste em um código Arduino/c++ utilizado no Wokwi com a placa ESP32. O código:
- `Conecta a placa à Internet via WiFi`
- `Faz equisição à API de Cotação da AwesomeAPI`
- `Exibe os valores, em R$, do Dólar Americano, EURO, Libra Esterlina e Dólar Canadense`

Caso queira testar o código no Wokwi, [clique aqui](https://wokwi.com/projects/447804235992971265) para executar a simulação!
